# Habr News Parser


**Сбор данных с Habr.com за считанные секунды.** Парсер автоматически извлекает заголовки, ссылки, авторов, даты публикации и тематические хабы из ленты новостей популярного IT-ресурса.

---

## 🎯 Для чего это нужно

Этот инструмент решает реальные задачи бизнеса и аналитики:

| Задача | Решение |
|--------|---------|
| **Мониторинг конкурентов** | Отслеживайте, какие темы обсуждают в IT-среде |
| **Анализ трендов** | Собирайте данные для выявления популярных направлений |
| **Поиск экспертов** | Находите авторов для сотрудничества по хабам |
| **Контент-план** | Изучайте, какие заголовки привлекают внимание |
| **Исследования рынка** | Формируйте базу для аналитических отчётов |

---

## ✨ Возможности

- ✅ **Парсинг нескольких страниц** — собирает данные с 1 до N страниц
- ✅ **Полная информация о статье**:
  - Заголовок
  - Прямая ссылка
  - Автор
  - Дата публикации
  - Все хабы (теги)
- ✅ **Автоматическое сохранение в CSV** — открывается в Excel, Google Sheets
- ✅ **Вежливый парсинг** — задержка между запросами (не нагружает сервер)
- ✅ **Обработка ошибок** — не падает при отсутствии данных
- ✅ **Русскоязычная кодировка** — корректное отображение текста в Excel

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий:

```bash
git clone https://github.com/VictoriSecret/hh-python-parser.git
cd hh-python-parser
```

### 2. Установка зависимостей:
```bash
pip install requests beautifulsoup4 pandas
```

### 3. Запуск парсера
```bash
python habr_parser.py
```
